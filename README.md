# NS Email Assets

Public static host for **Nice Shoes** transactional-email images (banners, headers).
Served via GitHub Pages so email clients can load them reliably.

Images live in `/img`. Public on purpose — email clients fetch images unauthenticated.
Do **not** put anything sensitive here.
